7.3 mss_disable_instance

(Ask a Question)

Description

This command is used to disable a core instance inside the MSS component.

mss_disable_instance \
-component_name {component_name} \
-instance_name {instance_name}

Arguments

ParameterTypeDescription
-component_name {component_name}stringMandatory. Name of the MSS component.
-instance_name {instance_name} stringMandatory. Name of the core instance to be disabled inside the MSS component.

Example

mss_disable_instance -component_name {test_sb_MSS} -instance_name {I2C_1}
