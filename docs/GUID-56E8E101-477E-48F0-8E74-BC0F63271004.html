8.33 remove_all_constraints

(Ask a Question)

Description

Removes all timing constraints from analysis.

remove_all_constraints

Supported Families

PolarFire®
PolarFire SoC
SmartFusion® 2
IGLOO® 2
RTG4™

Example

The following example removes all timing constraints from analysis.

remove_all_constraints
